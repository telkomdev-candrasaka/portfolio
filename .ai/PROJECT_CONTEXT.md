# Project Context

## Project Name

Candra Saka Rahardian Portfolio

## Purpose

Showcase backend engineering experience, case studies, projects, and technical writing.

The portfolio should communicate practical backend engineering ability, system design thinking, production experience, and full stack growth without becoming a generic developer portfolio.

## Target Audience

- Engineering Managers
- Senior Engineers
- Recruiters

## Main Sections

- Hero
- Experience Highlights
- Featured Case Studies
- Featured Projects
- Technical Expertise
- Articles
- Contact

## Guidance for Future AI Agents

- Keep the portfolio engineering-focused and content-driven.
- Prioritize case studies, backend architecture, system design tradeoffs, and project substance.
- Reuse existing data modules in `src/data/` instead of hardcoding portfolio content inside UI components.
- Reuse existing UI patterns and components before introducing new abstractions.
- Avoid unnecessary dependencies, large refactors, flashy animations, excessive gradients, and generic portfolio templates.
- Preserve accessibility, responsive design, strong TypeScript typing, and clear information hierarchy.
